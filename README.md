# Hello,

[m_atoi.asm](manual_func/m_atoi.asm) - This assembly program converts a string representation of an
integer into its numeric value, similar to the standard C library function
`atoi`.

[m_itos.asm](manual_func/m_itos.asm) - This assembly program converts an integer into its string

[m_stof.asm](manual_func/m_stof.asm) - This assembly program converts a string representation of a
floating-point number into its numeric value, similar to the standard C library
function `atof`.

These programs are designed for educational purposes and can be used to
understand low-level string manipulation and conversion techniques in assembly
language.
